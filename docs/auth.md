# Auth

LinkedIn session for read-only collection:

```powershell
npm run authLinkedIn
```

Greenhouse auth is archived and not needed for normal Job Radar runs:

```powershell
npm run authGreenhouse
```

Gmail auth is archived and not needed for normal Job Radar runs:

```powershell
npm run authGmail
```

Do not commit files from `storage/`, `credentials.json`, `gmail-token.json`, `.env`, resumes, cookies, or generated job data.
